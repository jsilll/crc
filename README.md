# Comparison of K-Cores algorithms

### Requirements

* [CMake](https://cmake.org/)
* [Boost](https://www.boost.org/)
* [GCC](https://gcc.gnu.org/) or [Clang](https://clang.llvm.org/)
* [Python](https://www.python.org/), for plotting the results

### Build

```bash
mkdir build
cd build
cmake ..
make
```

### Run

```bash
cd build/src
./<binary>
```
